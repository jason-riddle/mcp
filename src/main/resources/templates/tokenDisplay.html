Usage Examples

MCP Inspector

  1. Run: npx @modelcontextprotocol/inspector
  2. Set Transport Type to "SSE"
  3. Set URL to your MCP server (e.g., http://localhost:8080/mcp/sse)
  4. Click "Authorization" and paste your token in the "Bearer Token" field
  5. Connect and test your tools

Curl Example

Use the token in the Authorization header:

curl -H "Authorization: Bearer YOUR_TOKEN" http://localhost:8080/mcp/sse

MCP Client Configuration

For Claude Desktop or other MCP clients:

{
  "mcpServers": {
    "memory-server": {
      "url": "http://localhost:8080/mcp/sse",
      "headers": {
        "Authorization": "Bearer YOUR_TOKEN"
      }
    }
  }
}
